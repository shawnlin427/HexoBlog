开源软件合集

这个页面用于展示一些可用来替代对应商业软件的开源软件。此处的开源软件不排除双重许可的开源软件，即对个人免费但是对企业收费的开源软件。这里所列的开源软件一般是常用的软件，或者是生产力软件。

向开源者致敬！他们是互联网时代的国际主义者。

Firefox

说到开源当然少不了火狐。开源浏览器乃至开源软件中的标杆。

GIMP

类似PhotoShop的图像处理软件。

官网：https://www.gimp.org/

教程：https://www.bilibili.com/video/BV1bt411F7Ug?spm\_id\_from=333.337.search-card.all.click

FreeCAD

类似AutoCAD和Solidworks的工业三维建模软件。

官网：https://www.freecadweb.org/

教程：https://www.bilibili.com/video/BV1oM4y1P7DP?spm\_id\_from=333.337.search-card.all.click

Blender

类似3D Max的三维建模软件。

官网：https://www.blender.org/

教程：https://www.bilibili.com/video/BV1fb4y1e7PD?p=16 （针对最新的Blender3.0，好评）

OBS studio

录屏、直播推流软件，可以看到很多主播都用过。

官网：https://obsproject.com/

教程：https://www.bilibili.com/video/BV1XW411W7tH?spm\_id\_from=333.337.search-card.all.click

Kodi

开源的多媒体影音播放器。Kodi主要被作为家庭的影音中心被使用，可以通过Kodi把树莓派变成一个机顶盒。

官网：https://kodi.tv/

教程：http://www.kodiplayer.cn/course/

VLC

开源的多媒体播放器，类似Windows Media Player。

官网：https://www.videolan.org/vlc/index.zh\_CN.html

Seafile

开源的云盘。可以用Seafile搭建自己的网盘，但是得有自己的服务器

官网：https://www.seafile.com/download/

Shotcut Openshot Kdenlive

这三个都是开源的视频剪辑软件。Shotcut对新手更友好，当然功能也更简单。

Shotcut官网：https://shotcut.org/

OpenShot官网：https://www.openshot.org/zh-hans/

Kdenlive官网：https://kdenlive.org

Zotero

文献管理软件。我认为的少数开源吊打闭源（Endnote）的软件。

官网：https://www.zotero.org/

Python Julia

结合科学计算包可以替代Matlab的功能。但是Matlab的优势主要是在整合上（比如Symlink）。

Vosk

开源语音识别引擎，支持中文。

官网：https://alphacephei.com/vosk/

Mozilla Thunderbird

开源邮件管理软件。很可惜的是IOS端没有对应应用。
